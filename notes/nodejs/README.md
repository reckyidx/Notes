## 📚 How to Use This Material

### 1. **Start with Theory**
- Read theory files in order
- Each file builds on previous concepts
- Focus on understanding the "why" behind each concept

### 2. **Practice with Code**
- Run the example code in the `practice/` folder
- Modify and experiment with the examples
- Understand how each piece works

### 3. **Complete Exercises**
- Start with basic exercises
- Progress to intermediate and advanced
- Check your solutions against the `solutions/` folder

### 4. **Use the Cheat Sheet**
- Quick reference during interview preparation
- All key concepts and code examples in one place
- Perfect for last-minute review

## 🔧 Prerequisites

- Node.js v18+ installed
- npm or yarn package manager
- Basic JavaScript knowledge
- Understanding of asynchronous programming
- Familiarity with REST APIs
