# RabbitMQ Interview Questions for Senior Developers (10+ Years Experience)

This repository contains comprehensive and complex RabbitMQ interview questions tailored for senior developers with 10+ years of experience.

## Structure

- `01-architecture/` - Architecture and design pattern questions
- `02-message-patterns/` - Exchange types and routing questions
- `03-configuration/` - Configuration, tuning, and performance questions
- `04-security/` - Security and authentication questions
- `05-operations/` - Monitoring, operations, and troubleshooting questions
- `06-clustering/` - Clustering, high availability, and scaling questions
- `07-advanced/` - Advanced features and plugin questions
- `08-best-practices/` - Best practices and design patterns questions

## Usage

Each folder contains a markdown file with interview questions categorized by complexity level:
- **Intermediate**: Questions for experienced developers
- **Advanced**: Questions requiring deep technical knowledge
- **Expert**: Questions testing architectural thinking and system design
